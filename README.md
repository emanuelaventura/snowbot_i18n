## Quick start ##
# 这个文档展示了小雪机器人的功能和如何调用/二次开发这些功能




## 功能

小雪人SDK以aar形式集成功能包括以下：

1. 语音
    1.  语音识别 (Recognize) 
    2.  语义理解 (Understand)
    3.  语音合成 (Text To Speech)
2. 传感器
    1.  触摸传感器
3. 面部表情控制
    1.  表情控制
    2.  表情资源 
4. 底盘控制
    1.  运动控制
    2.  地图控制
5. 上身控制
    1.  手臂控制
    2.  触摸传感器
